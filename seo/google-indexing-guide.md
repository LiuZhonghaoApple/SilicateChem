# Google Indexing Guide — silicatechem.com

Operational guide for submitting, prioritizing, and monitoring Google indexing after deployment.

See also: [`indexing-guide.md`](./indexing-guide.md) for funnel layer context and technical SEO checklist.

---

## 1. Submit Sitemap in Google Search Console

1. Verify domain ownership at [Google Search Console](https://search.google.com/search-console).
2. Add property: `https://silicatechem.com` (domain property recommended).
3. Open **Sitemaps** in the left sidebar.
4. Enter `sitemap.xml` and click **Submit**.

**Live sitemap URL:**

```
https://silicatechem.com/sitemap.xml
```

Generated by `src/app/sitemap.ts` and referenced in `src/app/robots.ts`.

### Expected URL count (~30)

| Category | Count | Sitemap priority |
|----------|-------|------------------|
| Homepage | 1 | 0.95 |
| Money page (`/products/sodium-metasilicate`) | 1 | **1.0** |
| Other product pages | 4 | 0.9 |
| Product index | 1 | 0.85 |
| Guide pages | 5 | 0.8 |
| Guides index | 1 | 0.75 |
| Application pages | 4 | 0.6 |
| Applications index | 1 | 0.65 |
| About / Contact / FAQ | 3 | 0.7 |

**Excluded from sitemap (noindex):** `/blog`, `/blog/*`, `/factory`

### Post-submit checks

- Sitemaps → status **Success**
- `https://silicatechem.com/robots.txt` contains `Sitemap: https://silicatechem.com/sitemap.xml`
- `robots.txt` disallows `/api/` only

## 3. Request Manual Indexing (URL Inspection)

Use when a priority page is not indexed within 7 days of sitemap submission.

1. Google Search Console → **URL Inspection** (top search bar).
2. Paste full URL, e.g. `https://silicatechem.com/products/sodium-metasilicate`.
3. Wait for Google to test the live URL.
4. Confirm:
   - **Page is indexable** (no `noindex` on indexable pages)
   - **User-declared canonical** matches the inspected URL
   - **Sitemap** lists the URL
5. Click **Request indexing**.

**Limits:** Google may throttle manual requests (~10–20/day per property). Prioritize Tier 1–2 URLs first.

### Pages that should show "Excluded by noindex" (expected)

- `/blog` and `/blog/*`
- `/factory`

These remain crawlable via internal links but are not indexing targets.

## 4. Internal Link Funnel (Crawl Path)

Google discovers pages through internal links. The site is structured as:

```
Homepage → Products → Guides → Applications → RFQ (/contact)
                ↓
    /products/sodium-metasilicate (authority sink on every layer)
```

**Link hubs:**

| Component | Location |
|-----------|----------|
| `SiteExploreSection` | Homepage — full site map for crawlers |
| `ProductFunnelBanner` | Guides, applications, about, contact, FAQ |
| `FunnelLinksSidebar` | Guide and application detail pages |
| Footer | All pages — products, guides, applications |
| `InternalProductLinks` | FAQ sidebar |

Every indexable page should link to `/products/sodium-metasilicate`.

## 5. Monitoring After Launch

| Report | What to watch |
|--------|----------------|
| **Sitemaps** | Discovered vs submitted URL count |
| **Pages** | Indexed count growing; blog/factory excluded by noindex |
| **Performance** | Impressions on money page keywords first |
| **URL Inspection** | Re-check after major content updates |

Re-request indexing after:

- Money page content updates
- New guide or application pages
- Sitemap priority or URL structure changes
