## 4. Content pack (drafts)

All drafts are intentionally buyer-led and evidence-bounded. They should be fact-checked by a technical owner before any submission. A destination link is optional and must be editorially acceptable; it must never be inserted solely to force an anchor.

### Draft A — detergent / cleaning buyers

**Working title:** *A buyer's checklist for selecting sodium metasilicate in detergent formulations*
**Best-fit outlets:** ISSA, American Cleaning Institute
**Suggested destination:** `/blog/detergent-industry-metasilicate` or `/applications/detergent-industry`
**Suggested neutral anchor:** `detergent-industry metasilicate selection guide`

**Abstract (draft):**

Detergent buyers usually compare more than an assay number. A usable specification should state the grade and hydrate form, particle or solution handling requirements, packaging, batch documentation, storage controls and the supplier's response process for a deviation. This checklist separates the questions that can be answered from a technical data sheet from the questions that require a sample, application test or written commercial confirmation. It is written for formulators and procurement teams and does not prescribe a single supplier.

**Outline:**

1. Define the formulation job: alkalinity, builder contribution, processing aid or another documented function.
2. Match hydrate form and grade to the process; record the test method and acceptance range.
3. Check flow, moisture, dust, dissolution and packaging against the plant's equipment.
4. Request stable documentation (TDS/SDS and applicable certificates); keep batch-specific documents separate.
5. Agree sampling, non-conformance handling, traceability and change-notification expectations.
6. Compare total delivered risk, not only unit price; ask for a written quotation through the buyer's normal channel.

**Evidence links to cite:** the site's product specification/download path, application page, and the publication's own safety/handling references. Do not add an unverified regulatory or performance claim.

### Draft B — process / bulk-solids buyers

**Working title:** *Sodium metasilicate granules: a practical flow, dosing and storage checklist*
**Best-fit outlets:** Chemical Processing, Processing Magazine, Powder & Bulk Solids
**Suggested destination:** `/products/sodium-metasilicate-granules`
**Suggested neutral anchor:** `granular metasilicate handling checklist`

**Abstract (draft):**

Granular alkaline chemicals can fail a production plan through bridging, dust, moisture pickup or an unsuitable dosing point even when the nominal specification is acceptable. Before approving a source, the plant team can document receiving conditions, packaging integrity, transfer equipment, dust controls, storage residence time, sampling and a simple line trial. This checklist helps an engineer turn those observations into a comparable supplier request while keeping safety data and site procedures in control.

**Outline:**

1. Record receiving and storage constraints (covered area, humidity, pallet and bag handling).
2. Define transfer and dosing equipment, including the points where flowability matters.
3. Specify the required particle form and test method; do not infer it from a product name.
4. Plan a controlled trial with housekeeping, dust and dissolution observations.
5. Capture lot traceability, packaging photos and deviation actions.
6. Ask each supplier for the same fields so offers can be compared fairly.

### Draft C — application evidence note (for association / vertical outlets)

**Working title:** *How an industrial buyer can verify an alkaline chemical before scale-up*
**Best-fit outlets:** CISIA, TAPPI, Fibre2Fashion, WEF (only after application evidence is confirmed)
**Suggested destination:** the matching application page, not the homepage by default
**Suggested neutral anchor:** `application-specific procurement evidence`

**One-paragraph draft:**

The safest procurement decision is a documented fit decision: define the process objective, list the measurable acceptance criteria, run a controlled sample or line trial, and preserve the supplier's current technical and safety documents with the approval record. This approach lets a paper, textile, water-treatment or inorganic-silicate team reject a poor fit early without treating a marketing description as test evidence. Any application example should name its test scope and limits; results from one plant or grade must not be generalized to every process.
