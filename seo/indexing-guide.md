# Google Indexing Guide — silicatechem.com

Internal reference for submitting and prioritizing site indexing in Google Search Console.

---

## 1. Submit Sitemap to Google Search Console

1. Verify domain ownership at [Google Search Console](https://search.google.com/search-console).
2. Select the **silicatechem.com** property (URL-prefix or domain property).
3. Open **Sitemaps** in the left sidebar.
4. Enter `sitemap.xml` and click **Submit**.

The live sitemap URL is:

```
https://silicatechem.com/sitemap.xml
```

This is generated by `src/app/sitemap.ts` and referenced in `src/app/robots.ts`.

### After deployment

- Confirm `https://silicatechem.com/robots.txt` lists the sitemap URL.
- In Search Console → Sitemaps, status should show **Success** with the expected URL count (~30 URLs).
- Use **URL Inspection** on priority pages (below) and request indexing if not yet crawled.

---

## 2. Indexing Priority Layers

Submit and monitor pages in this order. Layer 1 pages carry the most commercial value and should be indexed first.

### Layer 1 — Money pages (highest priority)

| URL | Role |
|-----|------|
| `/` | Brand + manufacturer homepage |
| `/products/sodium-metasilicate` | **Authority sink** — primary RFQ conversion page |
| `/products/sodium-metasilicate-granules` | Primary export grade |
| `/products/sodium-metasilicate-anhydrous` | Grade specification page |
| `/products/sodium-metasilicate-pentahydrate` | Grade specification page |
| `/products/sodium-silicate` | Supporting product line |
| `/products` | Product catalog hub |

**Sitemap priority:** 0.9–1.0

### Layer 2 — Commercial guides (medium-high)

| URL | Role |
|-----|------|
| `/guides` | Guide index |
| `/guides/supplier-selection` | Supplier procurement intent |
| `/guides/price-factors` | Pricing / quotation intent |
| `/guides/uses-detergent` | Detergent industry intent |
| `/guides/sodium-metasilicate-vs-soda-ash` | Comparison intent |
| `/guides/how-to-choose-china-factory` | Factory verification intent |

**Sitemap priority:** 0.7–0.75  
All guides link to `/products/sodium-metasilicate` for conversion.

### Layer 3 — Applications (medium — discovery only)

| URL | Role |
|-----|------|
| `/applications` | Application index |
| `/applications/detergent-industry` | Mid-funnel use case |
| `/applications/water-treatment` | Mid-funnel use case |
| `/applications/textile-industry` | Mid-funnel use case |
| `/applications/paper-industry` | Mid-funnel use case |

**Sitemap priority:** 0.6–0.65  
**Note:** Application pages use `noindex` (mid-funnel only) but are included in the sitemap for crawler discovery and internal link equity. They are not SERP ranking targets.

### Layer 4 — Supporting pages

| URL | Role |
|-----|------|
| `/about` | Company credibility |
| `/factory` | Factory proof |
| `/faq` | Buyer questions |
| `/contact` | Inquiry / RFQ |
| `/blog` | Minimal supporting content |
| `/blog/*` | 3 supporting articles |

**Sitemap priority:** 0.45–0.7

## 3. Technical SEO Checklist

- [ ] `NEXT_PUBLIC_SITE_URL=https://silicatechem.com` set in production
- [ ] Canonical URLs on every page via `createMetadata()` in `src/lib/metadata.ts`
- [ ] OpenGraph + Twitter cards on all pages (default OG image: `/images/logo.svg`)
- [ ] `robots.txt` allows `/` and disallows `/api/`
- [ ] Homepage **Explore SilicateChem** section links to all indexable routes
- [ ] JSON-LD structured data present (Organization, Breadcrumb, FAQ where applicable)

---

## 4. Post-Launch Monitoring

1. **Coverage report** — watch for excluded pages; applications should show as "Excluded by noindex tag" (expected).
2. **Performance report** — track impressions/clicks for Layer 1 keywords first.
3. **Core Web Vitals** — ensure money pages pass CWV thresholds.
4. Re-request indexing after major content updates to money pages or guides.
