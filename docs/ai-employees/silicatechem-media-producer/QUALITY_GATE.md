# AI 物料制作质量门禁

## A. 事实

- [ ] 产品名称和 grade 正确
- [ ] CAS 和化学式正确
- [ ] 规格来自已确认资料
- [ ] MOQ、交期、产能和认证有来源
- [ ] 没有把推测写成承诺

## B. 真实证据

- [ ] 工厂/发货/客户视频来源明确
- [ ] 画面没有被 AI 改造成另一处场景
- [ ] AI 生成图已标记为解释性素材
- [ ] 图片符合 `trust-visual-allowlist.ts`
- [ ] 没有使用 stock、旅游、节日或无关活动图证明生产能力

## C. 隐私与授权

- [ ] 客户来访获得公开授权
- [ ] 人脸、车牌、订单号、合同和单据已处理
- [ ] 没有公开客户名称或未授权联系方式
- [ ] 业务人员确认可公开使用

## D. ICP 与 SEO

- [ ] 只对应一个 ICP
- [ ] 只对应一个主关键词
- [ ] 标题不是空泛宣传语
- [ ] Alt 文本描述真实画面
- [ ] 有对应产品页、文件页或 RFQ 链接
- [ ] CTA 清晰且与页面意图一致

## E. 技术

- [ ] 图片已压缩并有 WebP/AVIF 版本
- [ ] 视频有封面、字幕和文字稿
- [ ] 视频没有默认自动播放声音
- [ ] 文件名不含隐私信息
- [ ] 移动端尺寸已检查

## 结论

- `PASS`：可以进入人工发布审批
- `HUMAN REVIEW`：需要业务人员确认事实、授权或文案
- `BLOCKED`：存在虚构、隐私、事实冲突或素材缺失，禁止发布
